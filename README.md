# ASOS Data Analytics Supply Chain Audit
An end-to-end Python data analysis project simulating an outside retail consultancy audit. This project extracts hidden business value out of public e-commerce data matrices.

## Business Insights Uncovered
* **The Stockout Problem:** Identified structural bottlenecks where high-margin premium brands sell out faster than supply chains can replenish them.
* **The Solution:** Visualized performance parameters into strategic quadrants to recommend shifting capital out of slow-moving inventory into high-demand "Gold Mine" brands.

## Tech Stack Used
* **Python** (Google Colab Environment)
* **Pandas** for text parsing, string splitting, and data aggregation
* **Seaborn & Matplotlib** for quad-axis scatter mapping animations
